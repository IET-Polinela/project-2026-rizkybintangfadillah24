Website ini dibuat sebagai bagian dari praktikum Pemrograman Internet I menggunakan framework Django. Django adalah framework web berbasis Python yang digunakan untuk membangun aplikasi web dengan cepat, aman, dan terstruktur. Framework ini menyediakan berbagai fitur bawaan seperti sistem routing URL, template engine, dan pengelolaan database sehingga memudahkan pengembang dalam membuat aplikasi web. Website ini memiliki beberapa halaman seperti Home, About, dan Contacts yang saling terhubung.
